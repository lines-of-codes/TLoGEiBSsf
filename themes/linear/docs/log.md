| element | font-size (px) | line-height (px) | font-weight |
| ------- | -------------- | ---------------- | ----------- |
| h1      | 30             | 38               | 400         |
| h2      | 24             | 32               | 400         |
| h3      | 20             | 28               | 400         |
| p       | 16             | 24               | 400         |

while for posts on mobile devices:

| element | font-size (px) | line-height (px) | font-weight |
| ------- | -------------- | ---------------- | ----------- |
| h1      | 28             | 34               | 400         |
| h2      | 22             | 28               | 400         |
| h3      | 20             | 25               | 400         |
| p       | 16             | 22               | 400         |

> From [关于移动端各平台字号研究（px\pt\dp\sp）](https://www.zcool.com.cn/article/ZNjUwNTQ0.html).
